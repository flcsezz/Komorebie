# Komorebie Product Context

## Vision
- Komorebie is a premium deep-work sanctuary, not a generic productivity app.
- The experience should feel immersive, tactile, calm, and intelligent.

## Product Pillars
1. Immersive atmosphere
2. Low-distraction focus
3. AI-assisted studying
4. Meaningful progression
5. Convenient daily use

## Core Differentiators
- Interactive 3D environments instead of passive video backgrounds
- Luminous Glass visual system with depth and restraint
- Persistent AI sidekick for note understanding and flashcard generation
- 3D companion and room progression tied to real focus time
- Social presence that feels ambient, not noisy

## UX Priorities
1. Aesthetically premium
2. Not distracting during focus
3. Fast and convenient for repeat sessions
4. Clear enough for first-time users without onboarding fatigue

## Primary Modes
### Focus Mode
- Show only timer, current task/session, ambient controls, and emergency exit actions
- Hide progression, leaderboards, deep settings, and social surfaces
- Reduce motion intensity and contrast spikes

### Sanctuary Mode
- Show room, companion, customization, progression, and ambient social presence
- Support exploration before or after focus sessions

## Target Users
- Students doing long solo study sessions
- Knowledge workers seeking a premium focus environment
- Users who like gamification but reject childish UI

## Core Feature Areas
### Immersive Dashboard
- 3D study rooms
- High-fidelity timer
- Adaptive sound and music

### AI Study Sidekick
- Semantic note analysis
- Summary and focus task generation
- Automatic flashcards
- Focus coaching based on session behavior

### Sanctuary Progression
- 3D companion evolution
- Room customization
- Mana economy from completed focus time

### Social Layer
- Ambient friend silhouettes
- Real-time leaderboards
- Premium badges without cluttering focus flow

## Success Metrics
- 24-hour return rate after first session
- Average focus session length
- Flashcard generation usage rate
- Free-to-premium conversion for luxury environments and cosmetics
