# Design System: Zen System

## Brand Direction
- Premium, calm, minimalist, and restorative
- "Digital Zen Garden" aesthetic

## Color Foundation
- Deep Slate (Base): `#121212`
- Soft Charcoal (Surfaces): `#1E1E1E` / `#242424`
- Muted Sage (Focus/Success): `#B7C9B0`
- Deep Indigo (Indicators): `#4A5568`
- Negative space is a primary design element

## Typography
- Headers: Manrope (Clean, modern, approachable)
- Body: Inter (Functional, readable)
- High-contrast typographic scale

## Material Language
- Tonal layering instead of heavy shadows
- Glassmorphism for overlays: 20px blur, 50% opacity
- Soft, atmospheric glows for interactive elements

## Motion
- Easing must be slow and rhythmic, like a deep breath
- No "pop" or "bounce" effects
- Transitions should be smooth and subtle

## 3D & Scroll Experience
- Scroll-driven 3D environments (e.g., forest canopy, floating leaves)
- Parallax storytelling to create depth and immersion
- 3D elements anchor the "sanctuary" atmosphere

## Accessibility Constraints
- Preserve readable contrast on all surfaces
- Support reduced motion equivalents
- Use multiple indicators for state (not just color/glow)
