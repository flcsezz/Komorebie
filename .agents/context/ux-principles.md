# UX Principles

## Non-Negotiables
- The interface must feel premium but never busy.
- Focus mode must remove rather than merely dim secondary UI.
- Convenience beats cleverness for repeated user actions.

## Experience Rules
1. One dominant action per screen
2. Motion supports state change, never decoration alone
3. Social surfaces stay ambient unless explicitly opened
4. Gamification rewards should feel elegant, not arcade-like
5. AI should reduce effort, not create extra steps

## Focus Mode Rules
- Timer is the primary object
- Session task and progress are visible
- Ambient controls are reachable in one action
- Exit, pause, and resume are obvious
- No leaderboard, room store, or companion overlays by default

## Convenience Rules
- Start session in under 10 seconds
- Resume last environment and preferences automatically
- Upload notes with minimal ceremony
- Flashcards generated in the same study flow, not a separate maze
- Mobile and desktop layouts should preserve the same mental model

## Social Rules
- Presence should feel like shared silence
- Friend indicators default to subtle silhouettes or low-detail markers
- Social notifications never interrupt active focus mode

## Anti-Patterns
- Constant motion in peripheral UI
- Overuse of neon on dark backgrounds
- Multiple floating action surfaces at once
- Dense dashboard metrics competing with the timer
- Features locked behind confusing paywall interruptions
