# AetherStudy Agent Workspace

## Purpose
- Keep agent collaboration structured, low-noise, and resumable.

## Directories
- `context/`: stable source-of-truth docs
- `tasks/`: roadmap, task board, task templates
- `comms/`: decisions, handoffs, blockers, coordination notes

## Expected Loop
1. Read the relevant files in `context/`
2. Claim or update a task in `tasks/task-board.md`
3. Do the work
4. Record decisions or blockers in `comms/`
5. Leave a handoff note before stopping

## Priority Order
1. Aesthetic quality
2. Low-distraction focus mode
3. User convenience
4. Technical elegance
